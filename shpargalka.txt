план обучения котлин
1 - введение 
      среды разработки на Kotlin
Kotlin - это современный язык программирования, который стремительно набирает популярность. Он прекрасно подходит для разработки мобильных приложений под Android, веб-приложений, серверного кода и многого другого.
      Среды:
      1. Android Studio - это официальная IDE от Google, специально разработанная для создания Android-приложений. Она предоставляет все необходимые инструменты для разработки, отладки, тестирования и развертывания. 
      2. IntelliJ IDEA - это универсальная IDE от JetBrains, известная своей мощной функциональностью и гибкостью. 
      3. Visual Studio Code - это популярный редактор кода с открытым исходным кодом. Он известен своей легкостью, гибкостью и расширяемостью.
      4. Eclipse - это бесплатная и открытая IDE, которая традиционно использовалась для разработки на Java. 
2 - основы синтаксиса
      переменные - типы данных - операторы
В данном языке переменные объявляются с помощью ключевых слов "val" - неизменная переменная; или "var" - изменяемая переменная.
"val": значение такой переменной устанавливается только 1 раз при её инициализации. После этого неизменяется!
"var": зачение переменной можно изменять даже после ее инициализации. 
      Пример:
      val name = "Студент" // Неизменяемая
      var age = 23 // изменяемая 
      age = 27
в kotlin как и в других языках типы данных схожи, что позволяет быстро адаптироваться к работе:

      Short ()
